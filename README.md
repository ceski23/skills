# Skills Collection

A curated collection of AI agent skills for enhancing development workflows with expert guidance on modern frameworks, patterns, and best practices.

## 📚 Available Skills

### Vanilla Extract
**Comprehensive guide for styling React components with Vanilla Extract**

Master zero-runtime CSS generation with strong TypeScript ergonomics. This skill covers:
- Vite integration setup
- Typed variant systems (`styleVariants`, `recipes`)
- Design tokens and theming (`createTheme`, `createThemeContract`)
- Runtime variable updates (`@vanilla-extract/dynamic`)
- Global styles and utility-first patterns with `@vanilla-extract/sprinkles`

Perfect for building scalable, type-safe component styling systems.

### Preferred Libs
**Live recommender for choosing from your preferred GitHub library list**

Fetches `https://github.com/stars/ceski23/lists/standard-libraries` at runtime and recommends the best-fit library for a user's need using fresh repository evidence and recent commit inspection.

Ideal for questions like:
- Which preferred library should I use for routing, forms, tables, validation, animation, icons, email, or drag and drop?
- Pick the best option from my curated list for this feature
- Recommend a preferred package based on current maintenance signals

---

## 🚀 Installation

Add skills to your agent environment using:

```bash
bunx skills add https://github.com/ceski23/skills --skill vanilla-extract
bunx skills add https://github.com/ceski23/skills --skill preferred-libs
```

## 📖 Usage

Once installed, these skills provide domain-specific expertise and best practices for:
- Setting up and configuring technologies
- Implementing patterns and architectural decisions
- Code review and performance optimization
- Troubleshooting common challenges

The agent will automatically apply the appropriate skill when you ask questions or request work in these domains.
